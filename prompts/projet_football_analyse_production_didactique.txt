PROJET : ANALYSE DE PRODUCTION DIDACTIQUE – ENTRAÎNEMENT FOOTBALL

1.  Sujet proposé Conception d’une application d’aide à la décision pour
    le diagnostic des profils de performance d’une équipe de football et
    l’adaptation des contenus d’entraînement.

2.  Scénario du projet

-   Observation des matchs/situations de référence
-   Saisie des données dans une application
-   Scoring automatique selon référentiel football niveau 2
-   Diagnostic du profil d’équipe
-   Génération de recommandations d’entraînement
-   Réévaluation après cycle d’entraînement

3.  Benchmark Applications comparées :

-   Hudl
-   Wyscout
-   InStat
-   TacticalPad
-   Coach’s Eye

Valeur ajoutée : - Diagnostic automatisé - Recommandations
d’entraînement - Adaptation au milieu scolaire

4.  Applicabilité au milieu scolaire Utilisation en EPS pour :

-   diagnostic initial
-   différenciation pédagogique
-   évaluation formative et sommative
-   suivi de progression

5.  Coût technique Version prototype :

-   Excel / Google Sheets / formulaires
-   coût faible ou nul

Version avancée : - C# / MySQL / Web app - coût modéré

6.  Modèle de grille d’observation Critères :

-   efficacité collective
-   efficacité individuelle
-   rôle arbitre/observateur
-   connaissances conceptuelles

Résultat : - scoring /20 - diagnostic - recommandations d’entraînement
